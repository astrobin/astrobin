## Description

[AstroBin NG](https://github.com/astrobin/astrobin-ng).

## Installation

```bash
$ npm ci
```

## Running it

```bash
# development
$ npm run start

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm test

# test coverage
$ npm run test:cov
```
